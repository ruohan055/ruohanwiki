## **2.使用：**

**`F8` 呼出/隐藏菜单**

**方向键`↑`  `↓` 或者`8`  `2` 控制上下**

**方向键`←`  `→` 或者`4`  `6` 控制左右**

**`回车键/Ent/小键盘5` 确定； `退格键/Back/小键盘0` 返回**

****
